# Assets - Archivos estáticos

Este directorio contiene los archivos estáticos utilizados en la aplicación.

## Logo de la empresa

Para que el logo aparezca en los reportes PDF, coloca el archivo del logo en esta ubicación:

```
/static/assets/logo-kedikian.png
```

### Especificaciones del logo:
- **Formato**: PNG con fondo transparente
- **Dimensiones recomendadas**: 300x150px (o proporción 2:1)
- **Peso**: Máximo 500KB

### Uso actual:
- ✅ Reportes de Cuenta Corriente (PDF)

Si el archivo no existe, el PDF se generará sin el logo (no afectará el funcionamiento).
